Drop your animated background video here as **background.mp4**.

The component `<AnimatedBackground />` will look for `/background.mp4` and play it
autoplay / loop / muted / playsInline behind everything. If the file is missing,
a CSS cyberpunk gradient fallback renders in its place — the page still works.

Recommended file:
`Futuristic_cyberpunk_abstract_prediction_market_visualization_background...mp4`
(any 1080p or 720p mp4 with the cyber abstract look will work).
